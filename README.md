# FPGA-based-design--Traffic-controller-
An FSM traffic controller implemented on a Spartan‑7 FPGA using Verilog and Xilinx Vivado. It manages North‑South and East‑West signals with timed Green, Yellow, and Red transitions. A clock divider provides visible timing, LEDs show the lights, and a 7‑segment display shows the countdown. Verified through  hardware testing with  .xdc constraints.
